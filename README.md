# 💻 Códigos das Aulas de Programação — *Digital Humanities / Letras Clássicas (UFRJ)*

Este repositório reúne os **códigos produzidos ao vivo em aula** durante as atividades de programação ministradas para os alunos da disciplina **Digital Humanities** do **Programa de Pós-Graduação em Letras Clássicas da Universidade Federal do Rio de Janeiro (UFRJ)**.

As aulas tiveram como objetivo introduzir estudantes e pesquisadores de Humanidades ao pensamento computacional e às bases da programação moderna, utilizando **exemplos práticos e acessíveis**, sempre com foco em clareza, lógica e aplicabilidade no contexto das Humanidades Digitais.

---

## 📚 Conteúdo

O repositório está dividido em exemplos e exercícios produzidos **em dois formatos principais**:

- 🧩 **Portugol** — Introdução à lógica computacional, estruturas de decisão, laços de repetição e operadores relacionais.  
- 🐍 **Python** — Exercícios progressivos abordando:
  - Sintaxe básica e boas práticas de escrita de código;
  - Operadores lógicos e relacionais;
  - Estruturas condicionais e de repetição;
  - Criação e reutilização de **funções**;
  - Noções de **código limpo** e **organização modular**.

---

## 🧠 Objetivos de Aprendizagem

As atividades apresentadas em aula tiveram como metas principais:

- Introduzir **conceitos fundamentais de lógica de programação**;
- Desenvolver o raciocínio algorítmico a partir de problemas práticos;
- Demonstrar o uso do **Python** como ferramenta de pesquisa nas Humanidades Digitais;
- Incentivar a **escrita de código limpo e sustentável**, inspirada em boas práticas e obras de referência:
  - *Clean Code* — Robert C. Martin  
  - *Design Patterns* — Erich Gamma et al.  
  - *Clean Architecture* — Robert C. Martin  
  - *Software Architecture Fundamentals* — Neal Ford et al.

## 🧭 Como Usar

### 🔹 Executar os códigos em Portugol
1. Baixe e instale o **Portugol Studio**:  
   👉 [https://univali-lite.github.io/Portugol-Studio/](https://univali-lite.github.io/Portugol-Studio/)
2. Abra o programa e carregue o arquivo `.por` desejado.
3. Execute o código e acompanhe o fluxo lógico do programa.

### 🔹 Executar os códigos em Python
1. Instale o **Python 3.10+**:
   ```bash
   sudo apt install python3
   ```
2. Execute qualquer arquivo com:
   ```bash
   python3 nome_do_arquivo.py
   ```
